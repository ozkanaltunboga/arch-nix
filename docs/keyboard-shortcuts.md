## Hızlı Erişim

### Sistem

```bash
# Yeniden başlat
sudo reboot

# Kapat
shutdown now

# Uyut
systemctl suspend

# Kilitle
bash ~/.config/hypr/scripts/lock.sh
```

### Servisler

```bash
# Hyprland reload
hyprctl reload

# PipeWire restart
systemctl --user restart pipewire wireplumber pipewire-pulse

# NetworkManager restart
sudo systemctl restart NetworkManager

# Bluetooth restart
sudo systemctl restart bluetooth
```

### Debug

```bash
# Hyprland logları
journalctl --user -u hyprland -n 100

# Sistem logları
journalctl -xe

# GPU bilgisi
lspci -k | grep -A 2 -E "(VGA|3D)"

# Disk kullanımı
duf

# RAM kullanımı
btop
```

## İpuçları

### Çoklu Monitör

```bash
# Monitörleri listele
hyprctl monitors

# Workspace'i monitöre ata
hyprctl keyword workspace 1,monitor:HDMI-A-1
```

### Wallpaper

```bash
# Wallpaper değiştir
awww img ~/Pictures/Wallpapers/my-wallpaper.jpg --transition-type any

# Matugen çalıştır
matugen image ~/Pictures/Wallpapers/my-wallpaper.jpg
```

### Ses

```bash
# Ses cihazlarını listele
wpctl status

# Varsayılan cihazı değiştir
wpctl set-default @DEFAULT_AUDIO_SINK@

# EasyEffects aç
easyeffects
```

### Bluetooth

```bash
# Bluetooth manager
blueman-manager

# Manuel bağlan
bluetoothctl
> connect XX:XX:XX:XX:XX:XX
```

## Özelleştirme

Kendi kısayollarını eklemek için `~/.config/hypr/hyprland.conf` dosyasını düzenle:

```ini
# Yeni kısayol
bind = $mainMod, P, exec, my-custom-script.sh

# Mevcut kısayolu değiştir
bind = $mainMod, F, exec, chromium  # Firefox yerine Chromium
```
